# O'FAMM Website

Site web pour le cabinet O'FAMM - Obé Mawussé Fantodji

## Technologies

- Next.js 14
- TypeScript
- Tailwind CSS
- Swiper.js (pour les slides)
- Framer Motion (animations)

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Fonctionnalités

- Mode sombre/clair
- Slides interactifs
- Section admin pour gérer le contenu
- Design moderne et responsive
